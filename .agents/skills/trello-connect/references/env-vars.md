# Variables de entorno para skill `trello-connect`

## Requeridas

- `TRELLO_KEY`: API key de Trello.
- `TRELLO_TOKEN`: token de acceso generado para esa key.

## Opcionales

- `TRELLO_BOARD_ID`: board por defecto para comando `lists`.
- `TRELLO_LIST_ID`: lista por defecto para comando `cards`.

## Ejemplo

```dotenv
TRELLO_KEY=tu_key
TRELLO_TOKEN=tu_token
TRELLO_BOARD_ID=67f03e6fd2bde36d7cf8dbcf
TRELLO_LIST_ID=67f03f0cb9f81e0dac4f5456
```
